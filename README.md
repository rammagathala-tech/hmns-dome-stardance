# hmns-dome-stardance
Upgrades to the 16ft portable planetarium dome and development of an automated Show Maker pipeline for the NASA Stardance Challenge.

# HMNS Dome Upgrades & Show Maker in specific

## Track 1: Structural & Hardware Updates
* Updates on the 16-foot portable dome shell, exhaust/airflow mechanics, and hardware rigging.

## Track 2: Planetarium Show Maker
* Software tool to ingest NASA data and format it for the dome.
